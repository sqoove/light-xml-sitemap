Light-XML Sitemap dynamically creates dynamic XML Sitemap that comply with Google Sitemap protocol.

== Description ==

Light-XML Sitemap dynamically creates dynamic XML Sitemap that comply with Google Sitemap protocol. Upon activate, you will be able to select within few clicks which content must be include or exclude from the sitemap.

== Installation ==

1. Upload "light-xml-sitemap" folder to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Click the "Light-XML Sitemap" link located under your "Settings" section to configure the plugin options.
